# Security Policy

## Reporting a vulnerability

Please report security issues privately to **security@wexio.io** rather than
opening a public issue. Include a description, affected versions, and a minimal
reproduction if possible. We aim to acknowledge reports within a few business
days.

## Scope

This package is a client-side Tiptap extension: it highlights, autocompletes and
validates template text in the editor. It performs no network requests and runs
the `model-language` engine in-process. Language-engine issues should be reported
to [`model-language`](https://github.com/wexiohub/model-language).
